# Getting Started

Choose your path based on what you want to do:

## I want to build something

**[Hello World Demo](hello-world/)** — Build a Pomodoro timer from a
one-page brief to working code in 7 commands. Covers the full delivery
workflow: project setup, spec lifecycle, and implementation.

## I want to scope and estimate a project

**[Pre-Sales Demo](pre-sales-demo/)** — Take a sample RFP through the
pre-sales pipeline: synthesize client needs, generate a proposal with
proposed specs, and produce a SOW ready for delivery.

## Quick Command Reference

| Phase | Commands |
|-------|---------|
| **Pre-Sales** | `/ais.presales.synthesize` → `/ais.presales.propose` → `/ais.presales.scope` |
| **Setup** | `/ais.setup.plan` → `/ais.setup.architecture` → `/ais.setup.constitution` |
| **Spec Lifecycle** | optional `/ais.spec.brainstorm` → `/ais.spec.specify` → `/ais.spec.design` → `/ais.spec.tasks` → `/ais.spec.implement` |
| **Reporting** | `/ais.report.standup` · `/ais.report.status` · `/ais.report.project` · `/ais.report.metrics` · `/ais.report.retrospective` |
| **Maintenance** | `/ais.maintain.clarify` · `/ais.maintain.debug` · `/ais.github.sync` |

See the [full command reference](../reference/commands.md) for details.
