# AIS Brand Identity & Premium Design Standards

This skill is ALWAYS active when creating PowerPoint decks or Word documents. Every deliverable must reflect the AIS brand and achieve a polished, high-end consultative aesthetic.

## Brand Voice

- Concise, engaging, forward-thinking, mission/outcome focused
- First reference: "AIS" (never spell out "Applied Information Sciences" unless legally required)
- In titles, headings, and subheadings: use "&" instead of "and"
- Key differentiators to weave in: depth and breadth in transformative cloud technologies, Microsoft experience, outcome-focused delivery, collaborative customer relationships
- No generic filler. Every word earns its place.

## Color Palette

### Primary Colors (use these 80% of the time)

| Role | Hex | Usage |
|------|-----|-------|
| **Dark Navy** | `#1C2C57` | Primary background for dark slides, header strips, title bars. The dominant brand color. |
| **Medium Blue** | `#125A97` | Secondary blue for card headers, accent panels, right-side title panels |
| **AIS Orange** | `#F7901E` | Primary accent: top-edge bars, section labels, stat callout numbers, CTA buttons, vertical/horizontal accent lines. NEVER as a large fill area (except CTA buttons). |
| **White** | `#FFFFFF` | Card/tile backgrounds, title text on dark backgrounds |
| **Off-White** | `#F6F7F8` | Page background for light-mode content slides and document pages |
| **Dark Gray** | `#3C3C3C` | Primary body text on light backgrounds |

### Secondary Colors

| Role | Hex | Usage |
|------|-----|-------|
| **Muted Blue** | `#4F89C8` | Subtitle text on dark backgrounds, tertiary card accents |
| **Deep Navy** | `#1E3366` | Card backgrounds on dark slides (creates layered depth) |
| **Medium Gray** | `#6F7171` | Captions, compliance tags, partner labels, secondary text |
| **Light Blue Tint** | `#DEEBF8` | Sparingly, for secondary text on blue panels |
| **Light Gray** | `#EAEAE8` | Subtle borders, dividers on light backgrounds |

### Supplemental Colors (use sparingly)

| Hex | Notes |
|-----|-------|
| `#0B4468` | Deep teal-blue accent |
| `#006769` | Teal |
| `#118C8C` | Medium teal |
| `#58C4C5` | Light teal |

### Color Rules

- **Dark slides**: `#1C2C57` background, `#FFFFFF` titles, `#4F89C8` subtitles, `#F7901E` labels, `#1E3366` card tiles for depth
- **Light slides**: `#F6F7F8` background, `#1C2C57` header strip, `#3C3C3C` body text, `#FFFFFF` content cards with shadows
- **Orange is exclusively an accent** -- thin bars, labels, stat numbers, CTA buttons. Never fill large areas orange.
- **No gradients.** All fills are solid colors. Clean, flat design language.
- **Dark/light alternation**: Alternate between dark navy and light gray slides to prevent visual fatigue. Dark = bold and executive. Light = organized and readable.

## Typography

### Font Selection

| Priority | Font | When to Use |
|----------|------|-------------|
| **Primary** | Proxima Nova | When available (licensed premium font) |
| **Standard** | Arial | Default for all PowerPoints, Word documents, and email. Closest widely-available match to Proxima Nova. |
| **Premium decks** | Metropolis | For high-end consultative decks matching the AI Cloud Transformation style |

**Rule:** Use Arial by default. Use Metropolis when explicitly creating premium/consultative pitch decks. Never mix fonts within a single deliverable.

### Type Scale (PowerPoint)

| Role | Size | Weight | Color |
|------|------|--------|-------|
| Hero title (title slide) | 40-42pt | Bold | `#FFFFFF` |
| Section/slide title | 28-36pt | Bold | `#FFFFFF` |
| Page title (header strip) | 26-30pt | Bold | `#FFFFFF` |
| Stat/metric number | 36-48pt | Bold | `#F7901E` or `#125A97` |
| Card title / feature heading | 14-18pt | Bold | `#1C2C57` (light bg) or `#FFFFFF` (dark bg) |
| Subtitle / description | 13-15pt | Regular | `#4F89C8` (dark bg) or `#3C3C3C` (light bg) |
| Body text | 11-12pt | Regular | `#3C3C3C` (light bg) or `#4F89C8` (dark bg) |
| Section label / eyebrow | 9pt | Bold | `#F7901E`, ALL-CAPS |
| Caption / tag line | 8-9pt | Regular | `#6F7171` |
| CTA button text | 18-20pt | Bold | `#FFFFFF` |

### Type Scale (Word Documents)

| Role | Size | Weight | Color |
|------|------|--------|-------|
| Document title | 28-32pt | Bold | `#1C2C57` |
| Section heading (H1) | 20-24pt | Bold | `#1C2C57` |
| Subsection heading (H2) | 16-18pt | Bold | `#125A97` |
| Body text | 11-12pt | Regular | `#3C3C3C` |
| Caption / footnote | 9pt | Regular | `#6F7171` |

### Typography Conventions

- **ALL-CAPS**: Reserved for section/eyebrow labels (9pt bold orange), compliance tags, and category headers
- **Bold**: Titles, headings, stat numbers, step numbers, labels. Body text is regular weight.
- **Italic**: Used sparingly -- only for quotes or contrast emphasis
- **Left-aligned** for all body text and lists. Center only for: card titles in overview grids, stat numbers, CTA buttons, step numbers inside badges
- **Pipe separators** (`|`) for listing compliance frameworks, technologies, and partner details in captions

## PowerPoint Layout System

### Slide Dimensions

- **Canvas**: 13.33" x 7.5" (standard 16:9 widescreen). ALWAYS use widescreen.
- **Left margin**: 0.8" (consistent across all slides)
- **Content area width**: ~11.7" (0.8" left to ~12.5" right)

### Signature Design Elements

These elements are REQUIRED on every deck to maintain the AIS premium look:

1. **Orange top-edge accent bar**: Full slide width (13.33") x 0.06" tall, `#F7901E`, positioned at y=0. Present on ALL dark-background slides. This is the signature AIS decorative element.

2. **Dark navy header strip**: Full width (13.33") x 1.1" tall at top, `#1C2C57`. Used on all light-background content slides. Contains the slide title in white.

3. **Card-based information architecture**: Content organized into discrete, well-defined cards with consistent padding and shadows. No floating text without structure.

4. **Subtle card shadows**: `#000000` at 10% opacity, blur ~8pt, offset ~2pt at 225 degrees. Applied to all white content cards.

5. **Square corners**: No rounded corners on cards or shapes. All rectangles.

### Slide Types

#### Title/Cover Slide

- Full dark navy (`#1C2C57`) background
- Thin orange accent bar across top edge
- Left 2/3: eyebrow label (10pt bold gray), hero title (40-42pt bold white), subtitle (15pt muted blue), orange separator line (2.5" wide), compliance tags
- Right 1/3: Solid medium blue (`#125A97`) panel (~4.1" wide starting at x=9.0") with product/solution name centered

#### Header-Strip Content Slide (most common)

- Light gray (`#F6F7F8`) background
- Dark navy header strip: full width x 1.1" tall at top with slide title
- Content area below: cards, grids, text starting at ~1.3" from top
- White content cards with subtle shadows

#### Full Dark Section Slide

- Full dark navy background with orange accent bar at top
- Orange eyebrow label (9pt bold) at ~0.4" from top
- Large white title (30-36pt bold) below
- Muted blue description text
- Deep navy (`#1E3366`) content cards for stats or features

#### Closing/CTA Slide

- Full dark navy background with orange accent bar
- Centered large title (36-40pt bold white)
- Centered body text (14-15pt muted blue)
- Orange CTA button: `#F7901E` fill, white bold 18-20pt text, ~3.7-4.3" wide x 0.7" tall, centered
- Footer: "Applied Information Sciences | Microsoft Solutions Partner"

### Grid Systems

| Grid | Column Positions (x) | Column Width | Gutter | Use For |
|------|----------------------|-------------|--------|---------|
| **3-column** | 0.8", 4.9", 9.0" | 3.7" each | 0.4" | Capability cards, industry tiles, case studies |
| **4-column** | 0.8", 3.95", 7.1", 10.25" | ~2.8" each | 0.35" | Challenge cards, stat metrics |
| **2x3 grid** | 3 cols + 2 rows | 3.7" wide | 0.3" vertical | Industry verticals, guardrails, 6-tile grids |
| **Left-right split** | 60% left / 40% right | Variable | -- | Featured content + stats sidebar |

### Visual Elements

- **Orange horizontal separator**: 2.5" wide orange line on title slides below subtitle
- **Vertical orange accent bars**: 0.06" wide on left edge of white cards for emphasis
- **Colored card top-bars**: Thin colored bars atop card groups to differentiate pillars
- **Step number badges**: 0.7-0.8" squares in `#F7901E` (first step) or `#1C2C57` (subsequent) with centered white numbers
- **Icons**: Small PNG icons (0.38-0.5" square) as visual anchors in cards. Brand-colored icon circles (1.0" diameter) for overview cards.
- **No photography or illustrations** in premium pitch decks. Typography, color, and geometric shapes only. This avoids generic stock photography.

## Word Document Standards

### Page Setup

- US Letter (8.5" x 11")
- 1" margins all sides
- Font: Arial (or Metropolis for premium)

### Header

- AIS logo (white version on navy header bar, or color version on white)
- Thin orange accent line below header (`#F7901E`)

### Document Structure

- Title page: Dark navy background block with white title text, orange accent line, AIS logo
- Section headings: `#1C2C57` with orange accent underline or left-bar
- Body text: `#3C3C3C`, 11-12pt, generous line spacing (1.15-1.5)
- Tables: Navy header row (`#1C2C57`) with white text, alternating `#F6F7F8` / `#FFFFFF` body rows
- Callout boxes: White card with subtle shadow and orange left-border accent

### Footer

- Page numbers right-aligned, `#6F7171`
- "Applied Information Sciences" left-aligned in footer, `#6F7171`

## Premium Design Principles

These principles define the difference between a generic deck and an AIS premium deliverable:

1. **Strict color discipline**: Only brand colors. No rogue colors, gradients, or decorative patterns. Every color choice is intentional.

2. **Consistent type hierarchy**: One font family throughout. The eye always knows where to look first (title), second (subtitle), third (body).

3. **Restrained accent usage**: Orange appears exclusively as thin accent bars, labels, stat numbers, and CTA buttons. This restraint makes it pop.

4. **Card-based layout**: Content in discrete, well-padded cards with consistent shadows. Mirrors modern SaaS/product UI. Reads as contemporary and organized.

5. **Generous whitespace**: 0.8" margins, 0.25-0.3" card-internal padding, 0.4" column gutters. Every element breathes.

6. **Flat design**: No 3D effects, bevels, or textures. Depth conveyed only through color layering (navy > deep navy > white) and subtle 10% shadows.

7. **Dark/light rhythm**: Alternate between dark and light backgrounds across slides. This creates visual rhythm and prevents fatigue.

8. **Stat-forward data**: Large orange numbers (36-48pt bold) on dark backgrounds. Convey confidence in results.

9. **Numbered progression**: "01, 02, 03" step indicators with colored badges imply methodology and process rigor.

10. **No generic stock photos**: In premium consultative decks, rely on typography, color, and geometric shapes. If images are needed, use high-quality branded photography from the AIS SharePoint imagery library.

11. **No accent lines under titles**: Avoid decorative underlines beneath slide titles -- use whitespace or background color contrast instead.

12. **Content density balance**: Dense information, but never cramped. If a slide feels crowded, split into two slides rather than shrinking text.

## Quality Checklist

Before delivering any deck or document, verify:

- [ ] AIS logo present on title/cover and closing slides
- [ ] Orange top-edge accent bar on all dark-background slides
- [ ] Navy header strip on all light-background content slides
- [ ] Only brand colors used (no rogue colors)
- [ ] Single font family throughout (no mixing)
- [ ] Card shadows consistent (10% opacity, ~8pt blur)
- [ ] All corners square (no rounded corners)
- [ ] Generous whitespace (0.8" margins, proper gutters)
- [ ] Dark/light slides alternate for visual rhythm
- [ ] Stat numbers are large and bold (36-48pt)
- [ ] Section labels are 9pt bold orange ALL-CAPS
- [ ] No generic stock photography
- [ ] Footer with "Applied Information Sciences" on closing slide
- [ ] Text uses "&" in headings (not "and")
- [ ] AIS referenced as "AIS" not "Applied Information Sciences" in body copy
