# Style Catalog

Complete list of paragraph and character styles available in the AIS branded document template.

## Paragraph Styles (Content)

| Style ID | Style Name | Usage |
|----------|-----------|-------|
| `Normal` | Normal | Default body text |
| `FirstParagraph` | First Paragraph | Opening paragraph of a section (reduced top spacing) |
| `Heading1` | heading 1 | Top-level section headings (numbered in TOC) |
| `Heading2` | heading 2 | Subsection headings |
| `Heading3` | heading 3 | Sub-subsection / step headings |
| `Compact` | Compact | Tightly-spaced body text for step instructions |
| `ListParagraph` | List Paragraph | Bulleted or numbered list items |
| `BodyText` | Body Text | Body text with callout/note styling |
| `Body2` | Body 2 | Alternative body text variant |
| `SourceCode` | Source Code | Code blocks, command-line examples |
| `AISBoldBlue` | AIS Bold Blue | Bold blue accent paragraph |
| `AISBullets` | AIS Bullets | AIS-branded bullet list items |
| `FlushLeft` | Flush Left | Left-aligned, no indentation |
| `Parties` | Parties | Party/stakeholder identification |
| `SOWInstructions` | SOW Instructions | Statement of Work instruction blocks |

## Paragraph Styles (Structure)

| Style ID | Style Name | Usage |
|----------|-----------|-------|
| `Title` | Title | Document title (cover page) |
| `CoverTitle` | CoverTitle | Cover page title variant |
| `TOCHeading` | TOC Heading | Table of Contents header |
| `TOC1` | toc 1 | TOC level 1 entries |
| `TOC2` | toc 2 | TOC level 2 entries |
| `TOC3` | toc 3 | TOC level 3 entries |
| `Header` | header | Page header content |
| `Footer` | footer | Page footer content |
| `Caption` | caption | Image/table captions |

## Character Styles

| Style ID | Style Name | Usage |
|----------|-----------|-------|
| `Strong` | Strong | Bold emphasis within runs |
| `Hyperlink` | Hyperlink | Clickable URL text |
| `VerbatimChar` | Verbatim Char | Inline code / monospace text |
| `CommentReference` | annotation reference | Comment anchor markers |

## Table Styles

| Style ID | Style Name | Usage |
|----------|-----------|-------|
| `TableGrid` | Table Grid | Standard bordered table |
| `GridTable4-Accent1` | Grid Table 4 Accent 1 | Branded accent table |
| `GridTable5Dark-Accent1` | Grid Table 5 Dark Accent 1 | Dark header table |

## Notes

- **Heading1** is the primary section heading used for TOC generation
- **Compact** is the most-used style for step-by-step instructions (106 uses in source)
- **ListParagraph** handles bulleted content (226 uses in source)
- **FirstParagraph** provides the opening paragraph after a heading with appropriate spacing
- **SourceCode** renders with monospace font for code examples
