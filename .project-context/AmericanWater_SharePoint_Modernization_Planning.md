## About .potx Files

A `.potx` file is a **PowerPoint Office Theme template**. Unlike a regular `.pptx`, it:
- Defines reusable **slide masters, layouts, and themes**
- Contains **brand elements** (colors, fonts, logos, background graphics)
- Serves as the **template foundation** for creating consistent presentations
- May contain **placeholder slides** with instructional or sample content
